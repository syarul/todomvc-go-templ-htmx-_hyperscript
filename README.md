### Usage
- install go if you don't have
- run `go mod tidy` to fetch all needed modules
- install templ `go install github.com/a-h/templ/cmd/templ@latest`
- run `templ generate`
- finally run `go run .`
- alternatively you can compile into executable with `go build .`
